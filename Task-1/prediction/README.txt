
# 📈 Sales Prediction Using Linear Regression

🚀 Predict product sales based on advertising investments using Machine Learning and Linear Regression.

============================================================

📖 PROJECT OVERVIEW

This project uses Linear Regression to analyze advertising expenditures across:

📺 TV Advertising
📻 Radio Advertising
📰 Newspaper Advertising

and predicts future sales based on these marketing investments.

============================================================

🎯 PROJECT OBJECTIVES

✅ Analyze advertising data
✅ Discover relationships between advertising channels and sales
✅ Build a Linear Regression model
✅ Evaluate model performance
✅ Predict future sales using custom advertising budgets
✅ Generate actionable business insights

============================================================
